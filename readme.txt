1. La racine du projet doit être 'localhost\cars-comparer-2cs-project\'.
2. ajouter le fichier "tdw.sql" à PHPMyAdmin 
3. Afin de démarrer le projet, placez le dossier 'cars-comparer-2cs-project' dans \xampp\htdocs.
4. Une connexion à Internet est nécessaire pour que les éléments externes fonctionnent.
5. J'ai inclus des fonctionnalités facultatives, notamment :
   - Modification de la couleur principale du site.
   - Changement du logo, favicon, titre et description du site
   - formulaire contact us
   - Historique des reviews de l'utilisateur dans mon Profile
